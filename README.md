## Sobre o projeto

Este foi o meu primeiro projeto de programação, desenvolvido em 2024 para um trabalho da faculdade.

O projeto foi criado utilizando PHP, SQL, HTML, CSS e JavaScript, com foco em praticar conceitos de desenvolvimento web e integração com banco de dados.

Durante o desenvolvimento utilizei ferramentas de IA como apoio para aprendizado, pesquisa e aceleração do desenvolvimento, enquanto estudava PHP e banco de dados SQL paralelamente.

Mesmo sendo um projeto inicial, ele foi importante para desenvolver minha base em:
- lógica de programação
- CRUD com banco de dados
- estruturação de aplicações web
- integração frontend/backend
- modelagem básica de banco de dados
